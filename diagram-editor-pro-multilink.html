Shortcuts

  • Space + Drag: Pan canvas
  • Ctrl + Scroll: Zoom
  • Shift + Click: Multi-select
  • Ctrl+C/V: Copy/Paste
  • Delete: Remove selected
  • Enter: Edit text
  • G: Toggle grid
  • L: Link nodes mode
  • Arrow keys: Nudge nodes

Linking Nodes

  • Click "Link Nodes" or press L
  • Click the parent node first
  • Click the child node second
  • Right-click → "Link To..." for context menu
  • Right-click → "Unlink" to remove connection
